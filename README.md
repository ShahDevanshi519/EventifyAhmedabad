# Eventify

Eventify is a full-stack event management platform where users can explore events, book tickets, and manage their profiles.

## Features

### User Features
- User Registration
- Secure Login with JWT Authentication
- Forgot Password via Email
- Reset Password
- Change Password
- Edit Profile
- Browse Events
- Event Booking
- Rating & Feedback System

### Admin Features (Admin Panel In React)
- Admin Authentication
- Create Events
- Update Events
- Delete Events
- Manage Users
- View Bookings
- Dashboard Analytics

## Tech Stack

Frontend
- React.js
- Axios
- Tailwind CSS

Backend
- Node.js
- Express.js

Database
- MongoDB

Authentication
- JWT Access Token
- JWT Refresh Token

Other Tools
- Nodemailer
- Express-fileupload (Image Upload)
- CORS
- UUID etc.
